Movie App Documentation

  • GET List of movies

    End point: "/movies"

    HTTP Call Method: GET

    Request Body: None

    Response Body: Response Body: JSON object containing the movies

  • GET a single movie BY TITLE

    End point: "/movies/:[TITLE]"

    HTTP Call Method: GET

    Request Body: None

    Response Body: Response Body: JSON object containing genre, director, imageURL, featured/not featured of the movie

  • GET genre BY TITLE

    End point: "/movies/:[TITLE]"

    HTTP Call Method: GET

    Request Body: None

    Response Body: Response Body: A text message indicating genre of the movie

  • GET director info BY DIRECTOR_NAME

    End point: "/movies/:[DIRECTOR_NAME]"

    HTTP Call Method: GET

    Request Body: None

    Response Body: Response Body: JSON Object holding director information

  • Add new user

    End point: "/users]"

    HTTP Call Method: POST

    Request Body: 

    
                    {
                        "name": "John Doe",
                        "email": "jd@mail.com",
                        "favorite-movies": []
                    }

    Response Body: Response Body: 

    
                    {
                        "id": uuid,
                        "name": "John Doe",
                        "email": "jd@mail.com",
                        "favorite-movies": []
                    }

  • Change username

    End point: "/users/:[USERID]"

    HTTP Call Method: PUT

    Request Body: None

    Response Body: Response Body: Text message indicating the change of username
